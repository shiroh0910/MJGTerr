# MJGTerr

訪問先を地図上で管理するためのPWA（Progressive Web App）です。

## 機能

- Leafletを利用した地図表示
- 地図上へのマーカー追加・編集・削除
- Google Driveとの連携によるデータ保存
- オフライン対応

## 開発環境のセットアップ

1. 依存パッケージをインストールします。
   ```bash
   npm install
   ```

2. 開発サーバーを起動します。
   ```bash
   npm run dev
   ```

## メモ

テストユーザーは、Google Cloud PlatformのOAuth2テストユーザーに追加しておく
